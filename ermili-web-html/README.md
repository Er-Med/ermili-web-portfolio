# ermili.dev — Web Developer Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Brand identity

| Element | Value |
|---------|-------|
| Primary color | `#00d4aa` (electric teal) |
| Background | `#0a0e17` (deep navy) |
| Typography | Outfit + JetBrains Mono |
| Logo | "E" monogram mark + "ermili.dev" wordmark |

## Quick start

Open `index.html` in your browser, or run a local server:

```bash
npx serve .
```

## Customize

1. **Personal info** — Edit text in `index.html` (hero, about, stats).
2. **Projects** — Replace placeholder projects in the Work section with your real work.
3. **Links** — Update GitHub, LinkedIn, and email in the contact section.
4. **Colors** — Change CSS variables at the top of `css/style.css`.
5. **Contact form** — Wire up [Formspree](https://formspree.io) or your own API instead of the mailto fallback in `js/main.js`.
