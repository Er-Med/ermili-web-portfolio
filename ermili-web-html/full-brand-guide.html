04 — Signature Elements

What makes it unmistakably theirs.

Pick 2–3 to repeat consistently across site, CV, and social — these recur everywhere so they become recognizable on their own.

A

Typographic monogram

Initials as a custom lockup — doubles as favicon and loading-screen mark.

B

Custom cursor / hover signature

A trailing dot or magnetic hover state on every interactive element — felt, not just seen.

C

Grid / coordinate motif

Faint grid lines, corner crosshairs, coordinate labels — exposes structure on purpose.

D

Consistent motion curve

One easing curve, used everywhere, so movement itself becomes recognizable.
